# BanG Dream! Discography 查询 SOP

## When NOT to use

- 非 BanG Dream 企划相关
- Live / Event 票务场馆等信息（用 `bangdream-live`）
- News / 告知 / 资讯（用 `bangdream-news`）
- 官方 X 推文（用 `bangdream-tweet`）

## 查询流程

查询起点：

- Discography 页面（WordPress 站点）：https://bang-dream.com/discographies/

> **抓取优先级与 UA 门禁（决策层红线）**：
> - **优先用专用网页抓取工具**（webreader / Tavily 等：自带浏览器 UA、返回干净 markdown、省 token）；仅当环境没有这类工具，才降级到 `curl` / `py` 脚本。
> - **红线**：bang-dream.com 对 `/discographies/` 及所有详情页做 UA 门禁——裸 `curl`（默认 UA）返回 **404 空桩页**，易被误判成"JS 未渲染"。**任何抓取（含 curl/py 兜底）必须带浏览器 UA。**
> - 工具判定标准、示例名单、curl/py 骨架见 [网页抓取策略](#网页抓取策略)。

### 1. 从 Discography 列表页定位作品

访问 Discography 页面，根据用户描述，通过页面筛选功能筛选目标作品，调用参考 [Discography 页面筛选说明](#discography页面筛选说明)。

页面有分页逻辑（未筛选共 **24 页**），**务必翻完所有分页**，避免漏取。

列表卡片包含：
- 类型标签：CD / Blu-ray / 音楽配信 / LP / 夢ノ結唱
- 作品标题
- 発売日（可能为"2026年秋リリース予定"等未定占位）
- アーティスト（企划名）

每张作品的详细信息需跳转到卡片对应的**详情页**获取。

> **多企划合辑**（如「ガールズバンドパーティ！カバーコレクション」）会被**每个参与企划的 artist 筛选各自命中**——按某企划筛选时这类合辑也会出现，属正常，勿误判为"混入其他企划"。

### 2. 进入详情页采集文字

详情页 URL 形如 `https://bang-dream.com/discographies/<数字ID>/`（如 `/discographies/4178/`），**ID 为纯数字**（非 slug）。

Discography 详情页**文字信息充分，不需要读图**（ジャケット画像仅为商品视觉，文字已足够回答用户问题）。逐模块采集文字内容，详见 [详情页文字模块](#详情页文字模块)。

### 3. 识别详情页信息完整度

通过以下方式判断页面是否已完整更新：

**信息不完整的特征**：
- 発売日为"XXXX年秋リリース予定""2026年X月"等占位
- 収録内容写"詳細は続報をお待ちください""収録内容や予約開始時期などの詳細は続報をお待ちください"
- 无 価格 / 品番 / 購入リンク

**信息完整的特征**：
- 発売日为具体日期
- 有 収録内容（曲目 + credit）、特典、価格、品番 等

### 4. 详情页文字模块

详情页 meta 表 + 正文模块，全部以**文字**呈现，逐项采集：

<div id="详情页文字模块"></div>

#### メタ表（标题下方表格）
常见行（按页面实际出现的为准）：
- **発売日**：具体日期 或 占位
- **アーティスト**：企划名
- **商品タイプ**：如「シングルCD」「シングルCD＋Blu-ray」「Blu-ray」「音楽配信」；多盘位时按盘分列
- **価格**：按盘位列出（税込），如「Blu-ray付生産限定盤：9,900円(税込) 通常盤：1,540円(税込)」
- **品番**：按盘位列出，如「BRMM-11018 / BRMM-11019」

> 占位页可能只有 発売日 + アーティスト 两行，无 価格/品番。
>
> **meta 表渲染不稳定**：专用抓取工具（webReader）的 markdown **有时会丢掉整张 meta 表**（商品紹介文/収録内容/特典正常，但 発売日/価格/品番/商品タイプ 缺失）。若用户问 価格/品番/発売日 而 markdown 里没有，**用 curl/py 兜底取原始 HTML**（见 [网页抓取策略](#网页抓取策略)），从 `<th>/<td>` 表格行补齐——这些字段一定是服务端渲染的文字。

#### 商品紹介文
标题下的描述段落。

#### 収録内容 / 収録曲
- 曲目列表（编号 + 曲名）
- credit：作詞 / 作曲 / 編曲（配信曲常见）
- Blu-ray 类：含 live 本编 setlist（按 artist 分组列曲）、特典映像（如 バックステージコメント映像，标注「※特装版のみ」等限定条件）

#### 特典
- **初回生産分限定封入特典**：通常按盘位（特装版 / 通常版）分列，如 ジャケットステッカー、アナザージャケット 等
- **法人別特典**：按零售商逐家列出，常见商家：ブシロード オンラインストア、アニメイト、HMV、ゲーマーズ、楽天ブックス、タワーレコード、あみあみ、WonderGOO/新星堂、ソフマップ、TSUTAYA RECORDS、ディスクユニオン、とらのあな、Neowing、メロンブックス 等

#### 特製アイテム（如有，限定盤）
如 ライブフォトブックレット、キャンバスボードセット、特製BOX 等，常标注限定条件。

#### 購入 / 配信リンク（如有）
- 配信：`bmu.lnk.to/...` 等 Streaming/Download 链接（音楽配信类目必有）
- 購入：bushiroad-store.com（ブシロード オンラインストア）、bushiroad-music.com（商品情报页）等
- 同時購入キャンペーン 等活动说明（如有）

#### 関連アーティスト
页面底部关联企划（合辑可能列多个）。

## Output

只输出获取到的已确认信息。不输出信息完整度判断的过程。発売日/収録内容等为占位（未定）时，如实告知用户"官方暂未公布，待续报"。
